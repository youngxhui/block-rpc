# Message

自定义消息

# 消息结构

| 字段               | 长度（字节） | 描述                       |
| ------------------ | ------------ | -------------------------- |
| 魔数（Magic）      | 4            | 用于标识协议（如 0xB10CC） |
| 版本（Version）    | 1            | 协议版本（如 0x01）        |
| 消息类型（Type）   | 1            | 请求（0x01）或响应（0x02） |
| 数据长度（Length） | 4            | 数据部分的长度             |
| 数据（Payload）    | 可变         | 实际的数据内容             |
| 校验和（Checksum） | 4            | 用于校验数据的完整性       |
